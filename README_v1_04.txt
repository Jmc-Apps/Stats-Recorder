Goalie Stats Recorder v1.04

Built from v1.03.

Changes:
- Updated app version to v1.04.
- Master-data import now accepts Hockey Goalie Stats MasterData exports without requiring an exact source version.
- Import validation checks:
  sourceApp = Hockey Goalie Stats
  exportType = MasterData
  goalies array exists
- Displays source Hockey Goalie Stats version in Master Data status.
- Compatible with Hockey Goalie Stats v4.02 and future v4.x master-data exports.
